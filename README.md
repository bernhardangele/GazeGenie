# GazeGenie

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/bernhardangele/GazeGenie)

A versatile tool for parsing, cleaning, aligning, and analyzing fixations from eye-tracking reading experiments.

The easiest way to run this application is through GitHub Codespaces. This method requires **no local installation** and runs entirely in your browser.

3.  Once the terminal loads, the app will start automatically.

## Run via Docker

If you have Docker Desktop installed, you can run the app locally without configuring Python environments.

**1. Create a results folder:**

```bash
    mkdir results
```

**2. Run the container:**

    docker run --name gazegenie_app -p 8501:8501 -v $(pwd)/results:/app/results bangele1/gazegenie:latest

*Note: If you are on Windows Command Prompt, replace `$(pwd)` with `%cd%`.*

**3. Access the App:**
In your browser, navigate to: [http://localhost:8501](http://localhost:8501)

**4. Restarting the container later:**
If you stop the container, you can restart it without creating a new one:

    docker start -a gazegenie_app

## Local Installation (Python/Conda)

If you prefer to run the code directly on your machine, follow these steps.

### 1. Install Miniforge (for Mamba/Conda)
Download the installer for your OS (e.g., Windows):
[Download Miniforge3](https://github.com/conda-forge/miniforge/releases/latest/download/Miniforge3-Windows-x86_64.exe)

### 2. Environment Setup
Open your terminal (or Miniforge Prompt) and run:

    # Create the environment
    conda create -n eye python=3.11 -y

    # Activate the environment
    conda activate eye

    # Install system dependencies
    conda install conda-forge::cairo

    # Install Python requirements
    pip install -r requirements.txt

### 3. Run the Program
Whenever you want to run the app:

    conda activate eye
    streamlit run app.py

In your browser, navigate to: [http://localhost:8501](http://localhost:8501)
